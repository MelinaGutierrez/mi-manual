ELEMENTOS HTML

Elemento Descripcion Documentacion
<html> Representa la raiz del documento Documentacion
<body> Es el cuerpo del documento Documentacion
<div> Elemento generico Documentacion
<input> Crea controles interactivos para formularios basados en la web Documentacion
<head> Información general (metadatos) acerca del documento Documentacion
<header> Representa contenido introductorio, normalmente un grupo de ayudas introductorias o de navegación Documentacion
<span> Contenedor genérico en línea para redactar contenido, que no representa inherentemente nada Documentacion
<nav> Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación Documentacion
<aside> Representa una parte de un documento cuyo contenido está indirectamente relacionado con el contenido principal del documento Documentacion
<footer> representa un pie de página Documentacion
<main> Representa el contenido dominante del cuerpo de un documento Documentacion
<article> composición independiente en un documento Documentacion
<section> Representa una sección independiente genérica de un documento Documentacion
<h1-h6> Representan jerarquia en titulos siendo h1 la mas alta y h6 la mas baja Documentacion
<p> Distribuye el texto en párrafos Documentacion
<br> Produce un salto de línea en el texto Documentacion
<hr> Representa una ruptura temática entre elementos a nivel de párrafo Documentacion
<wbr> representa una oportunidad de salto de palabra Documentacion
<sup> Se usa para renderizar un texto un poco mas arriba que otro, por ejemplo un exponente Documentacion
<button> Elemento interactivo activado por un usuario con un mouse, teclado, dedo, comando de voz u otra tecnología de asistencia Documentacion
<a> crea un hipervínculo a páginas web Documentacion
<title> define el título del documento Documentacion
<sub> Especifica el texto en línea que debe mostrarse como subíndice únicamente por motivos tipográficos Documentacion
<sup> Especifica el texto en línea que se mostrará como superíndice únicamente por motivos tipográficos Documentacion
<strong> Indica que su contenido tiene una gran importancia, seriedad o urgencia. Documentacion
<em> Marca el texto que tiene énfasis. Documentacion
<mark> Representa texto que está marcado o resaltado para fines de referencia. Documentacion
<q> Indica que el texto adjunto es una cita breve en línea. Documentacion
<blockquote> Indica que el texto adjunto es una cita extendida. Documentacion
<cite> Se usa para describir una referencia a un trabajo. Documentacion
<dfn> Indica el término que se define dentro del contexto de una frase u oración de definición. Documentacion
<abbr> Representa una abreviatura o un acrónimo. Documentacion
<small> Representa comentarios secundarios y letra pequeña, como derechos de autor y texto legal, independiente de su estilo. Documentacion
<ins> Representa un rango de texto que se ha agregado a un documento. Documentacion
<del> Representa un rango de texto que se ha eliminado de un documento. Documentacion
<s> Muestra el texto tachado o con una línea que lo atraviesa. Documentacion
<code> Muestra su contenido con estilo para indicar que el texto es un fragmento corto de código de computadora Documentacion
<samp> Se utiliza para encerrar texto en línea que representa la salida de muestra de un programa de computadora Documentacion
<kbd> Representa un tramo de texto en línea que indica la entrada textual del usuario desde un teclado, voz o cualquier otro dispositivo de entrada de texto Documentacion
<pre> Representa texto preformateado que debe presentarse exactamente como está escrito en el archivo HTML Documentacion
<address> Indica que el HTML proporciona información de contacto para una persona o para una organización Documentacion
<img> El elemento HTML incrusta una imagen en el documento. Documentacion
<table> El elemento HTML representa datos tabulares, es decir, información presentada en una tabla bidimensional compuesta por filas y columnas de celdas que contienen datos. Documentacion
