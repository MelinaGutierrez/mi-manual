CSS

SELECTORES

Tipo Descripción Sintaxis Enlace
Universal (*) Selecciona todos los elementos. *{ declaraciones CSS } Ir a pagina
De Tipo (p) Permite seleccionar elementos usando el nombre del elemento. nombre_elemento { declaraciones CSS } Ir a pagina
De Clase (.foo) Permite seleccionar elementos en función de los valores del atributo class de los elementos. .valor_atributo { declaraciones CSS } Ir a pagina
De ID (#foo) Permite seleccionar un elemento en función del valor del atributo ID del elemento. #valor_atributo { declaraciones CSS } Ir a pagina
De Atributo (a[foo]) Permite seleccionar elementos en función de la presencia o del valor de un determinado atributo. selector[attr] { … } Ir a pagina
PSEUDO-CLASSES Permite seleccionar elementos en base a su estado (:disabled), posicion de los elementos en el documento(:nth-child) e interacción del usuario(:hover) selector:pseudo-class { … } Ir a pagina
PSEUDO-ELEMENTOS Los pseudo-elementos se añaden a los selectores y permiten añadir estilos a una parte concreta de los elementos seleccionados. selector::pseudo-elemento { … } Ir a pagina
COMBINACION DE SELECTORES Elementos Descendientes: Seleccionamos todos los elementos seleccionados por B que son descendientes de los elementos seleccionados por A A B {} Ir a pagina
Elementos Hijos: Seleccionamos todos los elementos seleccionados por B que son hijos de los elementos seleccionados por A A>B{} Ir a pagina
Elementos Hermanos: Seleccionamos todos los elementos seleccionados por B que son hermanos de los elementos seleccionados por A A~B {} Ir a pagina
